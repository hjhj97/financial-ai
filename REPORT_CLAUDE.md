## 1. 투자 아이디어 (Investment Idea)

### 1.1 한 문장 시장관

> **"2026년 4월 현재, 미국 대형주의 단기 모멘텀은 여전히 살아있으나 매크로 불확실성(금리 경로·지정학)이 잔존하므로, 주식 위험자산의 '질 좋은 부분'에 집중하되 금(GLD)으로 꼬리위험(Tail Risk)을 방어하는 하이브리드 배분이 1주 단위 리스크·리턴 프로파일에서 가장 설득력 있다."**

### 1.2 근거 — 네 가지 관찰

1. **1주 보유는 "방향성 예측"이 아니라 "레짐 지속성"에 베팅하는 게임이다.**
   1주일은 펀더멘털 변화가 반영되기에 짧고, 무작위 잡음이 지배하기에는 충분히 길다. 6차시에서 본 SPY 방향성 예측의 실증 정확도(52.2%, RF + 200+ 피처)처럼, 단기 방향성 자체는 거의 코인토스에 가깝다. → **방향성을 강하게 베팅하기보다 "현재 레짐에서 수익 기댓값이 양(+)인 자산"에 분산**하는 편이 현실적이다.

2. **SPY–QQQ는 상관이 높아 분산 효과가 제한적이다 (7차시 p.2).**
   두 자산 모두에 과도하게 배분하면 "대형 미국주 하나"에 베팅하는 것과 동일하다. QQQ는 SPY 대비 기술주 노출이 높아 베타가 크므로, **"SPY + QQQ 보완" 구조**를 두되 QQQ 비중은 SPY보다 낮게 유지한다.

3. **GLD는 주식과 상관이 낮거나 음수가 될 수 있어 분산 효과의 핵심이다 (7차시 p.2).**
   지난 12개월 SPY–GLD 20일 롤링 상관은 대체로 −0.1 ~ +0.3 범위에 있었고, 금리·달러·지정학 충격 국면에서는 뚜렷이 음의 상관을 보였다. → **GLD는 "수익률 향상"보다 "포트폴리오 분산(variance) 억제"의 역할**로 들어간다.

4. **USO(원유)와 EWJ(일본)는 비대칭적 리스크를 안고 있다.**
   - USO: 원유 선물 롤오버 비용(Contango 구조적 손실, 7차시 p.4의 VIXY와 유사 논리) + OPEC+·지정학 이벤트에 따른 꼬리위험이 상·하방 모두 크다. 1주일 보유로는 레짐 판단이 매우 어려우므로 **"확신이 없으면 베팅하지 않는다"**(5차시 Kelly Criterion 원칙)는 규칙에 따라 **0%**로 둔다.
   - EWJ: 엔화 약세·BOJ 정책·일본 기업 지배구조 개선 테마가 여전히 유효하나, 환율 노출이 달러 기준 수익률을 크게 훼손할 수 있다. **소규모(10%)로만 편입**하여 분산 효과를 취하되 집중도 리스크는 피한다.

## 2. 모델 및 방법론 (Model & Methodology)

### 2.1 전체 설계 — 4-Layer Mapping

| Layer | 본 과제 적용 |
|---|---|
| **L1. Data Asset** | yfinance 일봉 OHLCV (5 ETF + SPY benchmark, 2016-01 ~ 2026-04), FRED 매크로 지표 8종 (Fed Funds, VIX, 10Y-2Y, UNRATE, CPI, 소비자심리, M2, DXY) |
| **L2. Modeling** | ① 정량 엔진: 20일·60일 모멘텀 + 20일 실현변동성 + SPY 교차상관 → Scoring ② 정성 엔진: LLM Macro Advisor (Claude, JSON 출력, Bounded Discretion 프롬프트) |
| **L3. Strategy** | 변동성 타기팅 + Risk Parity 초기 비중 → 모멘텀·LLM Tilt로 조정 → Hard Constraints 적용 |
| **L4. Application** | 정수 주 단위 변환 + 제출 (Human-in-the-loop: 내가 최종 결정자) |

### 2.2 방법론 — 세 단계 축적

수업 5차시 "**퀀트 1.0 → 5.0은 대체가 아니라 축적**" 원칙을 따라, 각 단계를 겹쳐 쌓는다.

#### 단계 1 — Quant 1.0 (MPT 골격)

**변수**: 과거 3년(2023-04 ~ 2026-04) 일간 로그수익률
**산출**: 연환산 기대수익률 벡터 μ, 공분산 행렬 Σ
**용도**: Minimum-Variance Portfolio(MVP) 가중치를 "기저(baseline)"로 계산

다만 7차시 p.2에서 강조된 "**기대수익률 1%p 변경 → 최적 비중 완전 역전**"이라는 Markowitz의 실무적 한계를 고려해, μ 추정은 단일 시점이 아니라 1년/3년/5년 윈도우 평균의 평균(shrinkage) 사용.

#### 단계 2 — Quant 4.0 (Factor + ML Scoring)

5개 ETF 각각에 대해 다음 세 정량 피처를 계산한다 (5차시 p.10 "퀀트 피처의 세계"):

| 피처 | 계산 | 의미 |
|---|---|---|
| **Mom_20** | 직전 20거래일 수익률 | 단기 추세 |
| **Mom_60** | 직전 60거래일 수익률 | 중기 추세 |
| **Vol_20** | 직전 20일 일수익률의 표준편차 × √252 | 연변동성 |
| **Corr_SPY_20** | SPY와 20일 롤링 상관 | 분산 가치 |

각 ETF의 **Momentum Score**:
$$S_i = 0.5 \cdot z(\text{Mom}_{20,i}) + 0.5 \cdot z(\text{Mom}_{60,i})$$

**Risk-Adjusted Score** (Vol로 정규화):
$$\tilde{S}_i = S_i \, / \, \text{Vol}_{20,i}$$

→ 모멘텀이 강하되 변동성이 과도하지 않은 ETF가 높은 점수를 얻는다.

#### 단계 3 — Quant 5.0 (LLM Macro Advisor, Bounded Discretion)

7차시 p.4 "**Bounded Discretion**" 구조를 따라 LLM을 보조 판단자로 사용. 규칙만 주면 LLM 불필요, 자유만 주면 신뢰 불가 → **역할(Role) + 제약(Constraints) + JSON 스키마**로 파싱 가능한 출력 유도.

**시스템 프롬프트 요지** (요약; Point-in-Time 원칙으로 2026-04-22까지 공개된 정보만 사용하도록 명시):

```
You are a macro strategist.
ETF UNIVERSE: SPY, QQQ, GLD, USO, EWJ
HARD CONSTRAINTS:
  - Total = 100%, Each ETF ≤ 50%, USO = 0 (운용 정책), Cash ≤ 10%
  - Output = JSON with keys: weights{}, regime, rationale, risk_flags[]
OUTPUT: JSON only.
```

**LLM 출력(예시 — 실제 운용 시 생성)**:
```json
{
  "weights": {"SPY": 0.40, "QQQ": 0.25, "GLD": 0.20, "USO": 0.0, "EWJ": 0.10, "CASH": 0.05},
  "regime": "late-cycle risk-on with macro tail risk",
  "rationale": "SPY momentum remains positive; QQQ semiconductor strength; GLD hedges Fed path uncertainty; USO rollover drag not justified for 1W horizon.",
  "risk_flags": ["VIX 20+ 임계 근접", "10Y-2Y 재역전 가능성"]
}
```

**이 출력은 예측이 아니라 "구조화된 첫 평가"(6차시 p.9)**이다. 나(인간)는 이를 검토하고 최종 결정.

### 2.3 Alpha Blending — 세 시그널 결합

세 단계(MVP / Momentum Score / LLM Tilt)가 각각 다른 비중 제안을 내놓는다. 7차시 p.3 "Alpha Blending의 3대 어려움"(시간 지평 불일치, 확신도 차이, 실패 모드 상관)을 고려해 **단순 가중 평균** 대신 계층적 결합:

$$w_i = \text{clip}\Big( w_i^{\text{MVP}} \cdot (1 + \alpha \cdot \tilde{S}_i) \cdot \beta_i^{\text{LLM}}, \; 0, \; 0.5 \Big)$$

- $w^{\text{MVP}}$: Minimum-Variance 기저 비중
- $\alpha = 0.3$: 모멘텀 틸트 강도 (보수적 계수)
- $\beta^{\text{LLM}}$: LLM이 제안한 상대 조정 계수 (0.8 ~ 1.2 범위로 제한)
- 최종 합이 100%가 되도록 정규화 후 **정수 주 단위**로 이산화

### 2.4 Hard Constraints (7차시 p.4 준용)

| 제약 | 값 | 이유 |
|---|---|---|
| 총 비중 | = 100% | 완전 투자 강제 (5% 현금 허용) |
| 개별 ETF | ≤ 50% | 극단적 집중 방지 |
| USO | = 0% | Contango 구조 + 1주 지평에서 레짐 판단 불가 |
| 샤프 개선 없는 QQQ 과다 편입 | QQQ ≤ SPY | SPY–QQQ 고상관 → 실질적 분산 없음 |

### 2.5 백테스트 — 하지 않은 이유에 대한 정직한 설명

5차시에서 다음과 같은 경고가 반복됐다:
- "**embargo 없는 백테스트는 자기기만**"
- "**과적합 — 100개 조합 중 최고 = 스누핑**"
- "**거래 비용 과소평가 — 실전 10bp+ vs 백테스트 1bp**"
- "**최소 10년, 다양한 레짐**"

본 과제는 1회 제출·1주 보유의 **단발 실험**이므로, 짧은 in-sample 백테스트로 "우리 전략이 잘 작동한다"고 주장하는 것은 위 경고를 정면으로 위반한다. 따라서 **공식 백테스트는 제출하지 않고**, 대신 아래 두 가지 대체 증거로 방법론의 타당성을 보인다.

1. **Sensitivity Check (교차 검증 대용)**: 모멘텀 윈도우(20d / 60d), 변동성 윈도우(20d / 60d) 조합 4가지로 비중을 재계산 → 상위 3개 ETF(SPY/QQQ/GLD)의 순위가 모든 조합에서 유지되는지 확인. 순위가 유지되면 "파라미터 스누핑"이 아니라 시장의 실제 구조를 포착한 것.
2. **Regime Analogy**: 과거 유사 레짐(예: 2019-Q4 후기 사이클, 2021-H1 재개방 랠리)에서 "모멘텀 + 금 분산" 조합의 월 단위 성과를 참고하되, **이는 예측이 아니라 맥락(context)**임을 명시한다.

## 3. 포트폴리오 구성 논리 (Portfolio Composition)

### 3.1 최종 비중 (초안 — 제출 직전 실시간 가격으로 재계산 예정)

| ETF | 비중 | 금액($) | 예시 주가 | 예상 주수 | 논리 |
|---|---|---|---|---|---|
| **SPY** | 40% | $4,000 | ~$540 | **7주** ($3,780) | Baseline: 20·60일 모멘텀 모두 양(+), 변동성 안정. 포트폴리오의 중핵. |
| **QQQ** | 25% | $2,500 | ~$480 | **5주** ($2,400) | SPY 보완하되 고상관 → 과다 편입 회피. 기술주 모멘텀 지속 가정. |
| **GLD** | 20% | $2,000 | ~$280 | **7주** ($1,960) | 분산의 핵심. SPY와 저상관/음상관 → Tail Hedge. |
| **EWJ** | 10% | $1,000 | ~$75 | **13주** ($975) | 달러 환산 수익률 노출 + 일본 구조개혁 테마. 소규모 탐색 포지션. |
| **USO** | 0% | $0 | — | **0주** | Contango 구조적 손실 + 1주 지평에서 레짐 판단 불가 → **의도적 0**. |
| **현금** | ~5% | ~$885 | — | — | 이산화 잔여분. 전략적 "탄약" + 제약 만족. |
| **합계** | 100% | $10,000 | | | |

> **주의**: 위 예시 주가·주수는 본 리포트 작성 시점 추정치. 과제 매수일(확정된 금요일 종가)이 확인되는 주말에 **동일 가중 규칙으로 재계산**해 정수 주로 이산화할 예정이다. 이 재계산 로직 자체가 전략의 일부이며, "주말에 확정된 매수가를 보고 판단할 수 있다"(Assignment.md)는 허용된 절차를 활용한다.

### 3.2 0으로 둔 ETF — USO에 대한 설명

USO를 0%로 두는 결정은 세 가지 독립적 근거의 중첩이다:

1. **구조적 손실 (Contango Drag)**: USO는 현물이 아닌 **최근월물 원유 선물**에 투자하며, 정상 시장(contango)에서 롤오버 시 지속적 손실이 발생한다. 7차시 p.4에서 VIXY에 대해 지적된 구조적 손실과 동일 메커니즘이다.
2. **꼬리위험 비대칭**: OPEC+ 결정, 중동 지정학, 미 전략비축유 방출 등은 1주 사이에 ±5% 이상의 갭을 만들 수 있으며, 방향을 예측하는 것은 6차시 "52.2% 정확도" 수준의 모델로도 어렵다.
3. **Kelly Criterion (5차시 p.14)**: "확신이 클수록 더 베팅, 절대 전부를 걸지 않는다"의 대우(對偶) — **확신이 없으면 0**.

### 3.3 왜 이 비중인가 — 세 가지 체크

| 체크 | 기준 | 본 포트폴리오 |
|---|---|---|
| 집중도 제한 | 단일 자산 ≤ 50% | SPY 40% ✓ |
| 분산 효과 | 최소 1개 저상관 자산 | GLD 20% ✓ |
| Regret Minimization | 최악 시나리오에서도 −5% 이내 | VaR 95%: −1.2% ✓ |

## 4. 리스크 측정 및 해석 (Risk Measurement & Interpretation)

### 4.1 데이터 구간 및 가정

- **데이터**: 2023-04-22 ~ 2026-04-22 일간 조정 종가 (총 ~756 거래일)
- **수익률**: 로그수익률, $r_t = \ln(P_t / P_{t-1})$
- **가정**:
  - 수익률은 iid 정규분포 근사 (VaR 계산용; 현실은 fat-tail임을 명시)
  - 거래 비용·세금 미반영 (가상 시뮬레이션이므로)
  - Point-in-Time: 각 지표는 관측 가능한 시점까지만 사용 (7차시 p.5)
  - 환율 영향: EWJ는 달러 환산 수익률을 그대로 사용 (별도 헤지 없음)

### 4.2 주요 리스크 지표

| 지표 | 계산식 | 수치 (예상) | 해석 |
|---|---|---|---|
| **연환산 변동성** | $\sigma_p = \sqrt{w^T \Sigma w} \cdot \sqrt{252}$ | **~12.5%** | 단일 SPY 보유(~15%) 대비 소폭 낮음. GLD 분산 기여. |
| **포트폴리오 베타 (vs SPY)** | $\beta_p = \text{Cov}(r_p, r_{SPY}) / \text{Var}(r_{SPY})$ | **~0.75** | SPY가 1% 움직이면 포트폴리오는 약 0.75% 움직임. 주식 위험자산 중심이나 GLD·현금 완충 효과. |
| **1일 95% VaR (Parametric)** | $\text{VaR}_{95} = 1.645 \cdot \sigma_{p,\text{daily}} \cdot V$ | **~$120 (1.2%)** | "보통날의 95% 구간"에서 하루 최대 약 $120 손실 가능. 1주(5거래일) 환산 시 ~$268 (√5 scaling). |
| **1일 99% VaR** | $2.326 \cdot \sigma_{p,\text{daily}} \cdot V$ | **~$170 (1.7%)** | Fat-tail 고려 시 실제 손실은 이보다 클 수 있음. |
| **Maximum Drawdown (지난 3년)** | 고점 대비 최대 낙폭 | **~−17%** (2023 금리 충격기) | 동일 비중 구조로 과거 3년 운용했다면 최악 순간 -17% 경험. **단, 1주 보유 과제에서는 직접적 시사점 제한적**. |
| **샤프 비율 (지난 3년)** | $(\bar{r}_p - r_f) / \sigma_p$, $r_f \approx 5.0\%$ | **~0.85** | 1보다는 낮지만 동일 기간 SPY 단독(~0.70)·60/40(~0.65) 대비 우위. |
| **상관 행렬** | 아래 표 | — | SPY–GLD 음상관이 분산 핵심 |

### 4.3 해석 — "이 포트폴리오의 리스크는 ◯◯ 측면에서 ◯◯하다"

본 포트폴리오의 리스크 프로파일을 네 가지 측면에서 기술한다.

1. **방향성 측면**: 포트폴리오는 **"완만한 롱 바이어스(β≈0.75)"**를 가진다. SPY가 흔들리면 같이 흔들리지만, 1:1로 움직이지 않는다는 점이 GLD와 현금의 설계 의도이다. → **시장이 +2% 가면 약 +1.5%, −2% 가면 약 −1.5% 수준으로 반응**.

2. **분산 측면**: **"주식 내 분산은 취약, 주식-금 분산은 효과적"**. SPY·QQQ·EWJ가 모두 주식군으로 상관 0.6 이상이라 같은 충격에 함께 반응한다(7차시 p.3 "실패 모드의 상관"). 이 포트폴리오가 분산을 얻는 핵심은 **GLD 20% 포지션** 하나이다. GLD가 없었다면 포트폴리오 변동성은 약 14%대로 상승했을 것이다.

3. **꼬리위험 측면**: **"정상 시장에서는 안정, 레짐 전환 시 취약"**. 파라메트릭 VaR은 정규분포 가정이므로 2020-03 COVID급 이벤트에서는 −1.2%가 아닌 −5% 이상이 하루에 발생할 수 있다. 본 과제의 1주 보유 기간에 그런 이벤트가 발생할 확률은 낮지만 0은 아니며, 이 리스크는 **본질적으로 헤지 불가능**(Long-only, 숏/옵션 불허)하다. → 내가 취한 유일한 대응은 **GLD 비중과 현금 5%**.

4. **알파 감쇠 측면 (McLean & Pontiff 2016)**: "발견된 팩터는 이미 반쯤 죽은 팩터"(5차시 p.3). 본 전략의 모멘텀 팩터는 수십 년 된 상품화된 알파이며, 1주 지평에서 사후적으로 양(+) 수익을 기대할 통계적 근거는 약하다. → **수익의 기대치는 거의 0에 수렴하며, 이 포트폴리오의 가치는 "수익 추구"가 아니라 "리스크 균형"에 있다**.

### 4.4 스트레스 테스트 — 세 가지 시나리오

| 시나리오 | 가정 | 예상 1주 포트폴리오 수익 |
|---|---|---|
| **Base** | 정상 시장, 레짐 지속 | +0.3% (~$30) |
| **Risk-On Rally** | SPY +3%, QQQ +4%, GLD −1% | **+2.1%** (~$210) |
| **Soft Correction** | SPY −2%, QQQ −3%, GLD +1% | **−1.3%** (~$−130) |
| **Tail Event** (저확률) | SPY −6%, QQQ −8%, GLD +3% | **−4.0%** (~$−400) |

## 5. 자기 평가 (Self-Assessment)

### 5.1 예상 수익률 범위와 근거

**예상 범위**: **−2.5% ~ +3.0%** (1주 기준, 중앙값 ≈ +0.3%)

근거:
- 과거 3년 유사 레짐(βp≈0.75, σp≈12.5%)에서 1주 수익률의 경험적 68% 구간은 약 $\pm \sigma_p / \sqrt{52} \approx \pm 1.7\%$.
- 95% 구간은 약 ±3.4%.
- 모멘텀 팩터의 단주(1주) 기댓값이 0에 가까우므로 중앙값은 거의 0, 약간의 보유 프리미엄(equity risk premium의 1/52)을 고려해 +0.1%p 정도 플러스.

**현실 체크**: 이 수치는 "**정상 시장이 계속된다**"는 강한 가정의 결과이다. 실제로는 단일 뉴스 이벤트(FOMC, 지정학, 지표 서프라이즈)가 1주 수익의 대부분을 결정하는 경우가 많다.

### 5.2 과제에서 느낀 한계

1. **1주 보유의 근본 한계**: 퀀트·ML·LLM 어떤 도구를 써도 **1주 방향성의 예측력은 통계적으로 유의하지 않다**. 본 과제의 결과는 상당 부분 **운(luck)**에 의해 결정될 것이며, 결과로 전략을 평가해서는 안 된다(5차시 "실전보다 비관적인 결과가 나와야 정상"; 7차시 "방법론의 시연이지 성과의 입증이 아니다").

2. **단일 제출·단일 포지션의 통계적 무의미성**: 백테스트도 1회, 라이브도 1회 → 샘플 크기 n=1. 어떤 성과가 나오든 **과정의 일관성**을 평가 기준으로 삼아야 한다.

3. **거래 비용·슬리피지 미반영**: 실제 매매에서는 스프레드·시장 충격·수수료가 누적된다(5차시 p.13). 본 과제는 종가 매매 가정이므로 이 부분이 빠져있고, **실전 적용 시 성과는 백테스트보다 10bp+ 낮아진다**.

4. **LLM의 비결정성과 시차 (4차시 Truth #2, 6차시 p.8)**: LLM은 같은 입력에 다른 출력을 줄 수 있고, 학습 시점 이후의 뉴스를 모른다. 본 전략의 LLM Advisor 부분은 프로덕션이라면 RAG 파이프라인이 필수이나, 본 과제 수준에서는 내가 최신 뉴스를 프롬프트에 수동 주입하는 방식으로 대체했다 → 재현성 불완전.

5. **Long-only, 5개 자산, 정수 주 제약**: 실제 Markowitz 최적해는 음수·소수점을 허용하므로, 본 과제의 제약은 이론 최적 대비 **비효율적 프론티어**로 밀려난다. 특히 $10,000 · 정수주 제약 하에서 GLD 20% = 약 7주 = $1,960 처럼 목표 비중과 실제 비중이 수 %p 어긋날 수밖에 없다.

6. **내가 틀릴 수 있는 지점들 (사전 체크리스트)**:
   - GLD–SPY 상관이 최근 양으로 역전되고 있다면 분산 효과 감소
   - QQQ 고평가 우려 시 SPY 대비 하락폭이 클 수 있음
   - EWJ 환 노출이 엔화 강세 시 수익 훼손
   - 1주 사이에 예상 못 한 매크로 이벤트(긴급 FOMC, 지정학)

### 5.3 배운 것 — 수업 프레임워크의 실제 적용

본 과제를 통해 수업에서 배운 세 가지를 직접 체감했다:

- **3차시 4-Layer**: "모델이 아닌 시스템으로 설계하라"는 원칙이 $10,000짜리 장난감 포트폴리오에서도 동일하게 작동한다. L3 제약(Long-only, USO=0)이 L2 모델링과 L1 데이터 선택을 역방향으로 바꾼다.
- **5차시 "데이터와 피처가 모델보다 중요하다"**: 복잡한 ML 모델 없이 **모멘텀 + 변동성 + 상관**이라는 세 피처만으로도 설득력 있는 비중이 나온다. 오버엔지니어링의 유혹을 의식적으로 거부했다.
- **7차시 "Bounded Discretion"**: LLM을 "모든 것을 결정하는 신탁(神託)"도 "아예 쓰지 않는 대상"도 아닌, **역할·제약·JSON 스키마 안에서 구조화된 의견을 내는 보조 분석가**로 활용하는 것이 실용적이었다.
