## 1. 투자 아이디어

### 1.1 시장관

2026년 4월 현재 미국 대형주의 단기 모멘텀은 여전히 양(+)이나, 매크로 불확실성(금리 경로·지정학)이 잔존한다. 1주 지평에서는 **레짐 지속성**에 베팅하되, 주식 내 집중보다 **SPY–GLD 저상관**을 활용한 분산이 리스크·리턴 효율을 결정한다.

### 1.2 네 가지 관찰

1. **1주 보유는 방향성 예측이 아니라 레짐 베팅**이다. → 강한 방향 베팅 회피, 레짐 내 양(+) 기댓값 자산에 분산.
2. **SPY–QQQ 고상관** (ρ ≈ 0.92) → 두 자산을 동시에 크게 담으면 단일 베팅과 동일. QQQ ≤ SPY 제약.
3. **SPY–GLD 저상관/음상관** (ρ ≈ 0.05) → GLD는 수익 추구가 아닌 **분산 엔진**으로 편입.
4. **USO·EWJ는 비대칭 리스크**: USO는 Contango 구조적 손실 + 지정학 꼬리위험, EWJ는 환노출. USO는 **0**, EWJ는 소규모(10%).

## 2. 모델 및 방법론

### 2.1 4-Layer Framework 매핑 (3차시 프레임워크 준용)

| Layer | 본 과제 적용 |
|---|---|
| **L1. Data Asset** | yfinance 일봉 OHLCV (5 ETF + SPY 벤치마크, 2023-04 ~ 2026-04), FRED 매크로 8종 (VIX, Fed Funds, 10Y-2Y, CPI, UNRATE 등) |
| **L2. Modeling** | 정량 엔진(모멘텀·변동성·상관 Score) + 정성 엔진(LLM Macro Advisor, JSON 출력) |
| **L3. Strategy** | 3-레짐 필터 → Inverse-Vol 초기 배분 → Alpha Blending → Hard Constraints |
| **L4. Application** | 정수 주 이산화 · Human-in-the-loop 최종 결정 |

### 2.2 3단계 축적 (Quant 1.0 → 5.0, 5차시 원칙)

**단계 1 — MPT 골격 (Quant 1.0)**
과거 3년 일간 수익률로 공분산 Σ 추정, **Minimum-Variance Portfolio(MVP) 비중을 기저(baseline)**로 사용. 기대수익률 μ는 1/3/5년 윈도우 평균의 shrinkage (7차시 p.2 "1%p 변경 → 최적 비중 역전" 문제 완화).

**단계 2 — Factor Scoring (Quant 4.0)**

각 ETF에 대해 네 피처 계산 (5차시 p.10):

| 피처 | 의미 |
|---|---|
| Mom_5d, Mom_20d | 단기/중기 추세 |
| Vol_20d | 연변동성 (리스크) |
| Drawdown_20d | 최근 고점 대비 낙폭 (반등 vs 붕괴) |
| Corr_SPY_20d | 분산 가치 |

**점수식**:
```
trend_i     = 0.6 · z(Mom_5d_i) + 0.4 · z(Mom_20d_i)
risk_pen_i  = 0.6 · z(Vol_20d_i) + 0.4 · z(Drawdown_20d_i)
total_i     = trend_i − risk_pen_i
```
→ 추세가 좋되 변동성·낙폭이 통제된 ETF가 높은 점수.

**단계 3 — LLM Macro Advisor (Quant 5.0, Bounded Discretion)**

7차시 p.4 "규칙만 주면 LLM 불필요, 자유만 주면 신뢰 불가 → **Role + Constraints + JSON**" 원칙.

```
SYSTEM: You are a macro strategist.
UNIVERSE: SPY, QQQ, GLD, USO, EWJ
HARD CONSTRAINTS: Total=100%, Each≤50%, USO=0, Cash≤10%
OUTPUT: JSON { weights, regime, rationale, risk_flags[] }
POINT-IN-TIME: Use only info available by decision date.
```

출력은 예측이 아니라 **구조화된 첫 평가**(6차시 p.9). LLM 제안은 정량 엔진 비중을 ±5~10%p 범위에서만 조정하도록 제한.

### 2.3 3-레짐 필터

| 레짐 | 조건 | 편향 |
|---|---|---|
| **Risk-On** | SPY·QQQ 20d 모멘텀 > 0, 변동성 안정 | SPY/QQQ 중심, GLD 소량 헤지 |
| **Mixed / Uncertain** | SPY·QQQ 방향 엇갈림 또는 변동성 급등 | GLD 확대, QQQ 축소, EWJ 분산 보완 |
| **Commodity / Defensive** | 주식 모멘텀 둔화, GLD·USO 상대 강세 | 주식 축소, GLD 확대 |

**현재 판단**: Risk-On ~ Mixed 경계. 주식 우위 유지하되 GLD 분산 비중을 평상시보다 높게 설정.

### 2.4 Alpha Blending (7차시 p.3)

세 시그널을 계층적으로 결합 (단순 평균 회피):
```
w_i = clip( w_MVP_i · (1 + α · total_i / vol_i) · β_LLM_i ,  0,  0.5 )
```
- α = 0.3 (보수적 모멘텀 틸트), β_LLM ∈ [0.8, 1.2]
- 최종 합 = 100%가 되도록 정규화 → 정수 주 이산화

### 2.5 Hard Constraints (7차시 p.4)

| 제약 | 값 | 근거 |
|---|---|---|
| 총 비중 | = 100% (현금 ≤ 10%) | 완전 투자 강제 |
| 개별 ETF | ≤ 40% | 극단적 집중 방지 |
| SPY + QQQ | ≤ 55% | 미국 주식 단일 베팅화 회피 |
| USO | = 0% | Contango + 1주 레짐 판단 불가 + Kelly 원칙 |
| 선택 ETF 수 | 2 ~ 4개 | 분산 요건 |

### 2.6 백테스트를 제출하지 않는 이유 (방법론적 정직성)

5차시에서 반복 경고: "embargo 없는 백테스트는 자기기만", "100개 조합 중 최고 = 스누핑", "최소 10년·다양한 레짐 필요". 본 과제는 n=1 단발 실험이므로 짧은 in-sample 백테스트로 전략 타당성을 주장하는 것은 원칙 위반이다. 대신 **Sensitivity Check**(모멘텀/변동성 윈도우 4조합에서 상위 3개 ETF 순위 유지 확인)로 파라미터 스누핑이 아님을 보인다.

### 3.3 USO = 0 결정의 삼중 근거

1. **구조적 손실**: USO는 최근월물 선물 → 정상 시장(contango)에서 롤오버 손실 (7차시 p.4 VIXY 논리).
2. **꼬리위험 비대칭**: OPEC+·지정학·전략비축유 → 1주 사이 ±5%+ 갭. 방향 예측력은 52% 수준 모델로도 불가.
3. **Kelly Criterion** (5차시 p.14): "확신이 클수록 베팅, 절대 전부 걸지 않는다"의 대우 — **확신 없으면 0**.

### 3.4 Regret Check

- 단일 자산 ≤ 50% ✓ (최대 SPY 40%)
- 최소 1개 저상관 자산 ✓ (GLD 20%)
- 최악 시나리오 −5% 이내 ✓ (99% VaR ≈ −1.7%)

## 4. 리스크 측정 및 해석

### 4.1 데이터 구간과 가정

- **기간**: 2023-04-22 ~ 2026-04-22 일간 조정 종가 (~756 거래일)
- **가정**: 수익률 iid 정규 근사 (현실은 fat-tail — 한계 명시), 거래비용·세금 미반영, Point-in-Time 준수, EWJ 환 노출 미헤지

### 4.2 주요 지표 (예상)

| 지표 | 수치 | 해석 |
|---|---|---|
| **연환산 변동성 σ_p** | **~12.5%** | SPY 단독(~15%) 대비 낮음. GLD 분산 기여. |
| **포트폴리오 베타 (vs SPY)** | **~0.75** | SPY 1% 시 포트폴리오 0.75% 반응 — 완만한 롱 바이어스. |
| **1일 95% VaR** | **~$120 (−1.2%)** | 정상 시장 하루 최대 손실. 1주 환산 ~$268. |
| **1일 99% VaR** | **~$170 (−1.7%)** | Fat-tail 시 실제 손실은 더 클 수 있음. |
| **Maximum Drawdown (3Y)** | **~−17%** | 동일 비중을 3년 유지했다면 최악 순간. 1주 과제와 직접 연결은 제한. |
| **샤프 비율 (3Y, r_f=5%)** | **~0.85** | SPY 단독(~0.70)·60/40(~0.65) 대비 우위. |

### 4.4 네 측면 해석

1. **방향성 측면** — **완만한 롱 바이어스 (β≈0.75)**. 시장 +2% 시 ≈+1.5%, −2% 시 ≈−1.5%. GLD·현금이 반응 탄력을 낮춤.
2. **분산 측면** — **"주식 내 분산은 취약, 주식-금 분산은 효과적"**. SPY·QQQ·EWJ 모두 ρ≥0.6 → 같은 충격에 함께 반응 (7차시 "실패 모드의 상관"). 분산 효과의 90%는 GLD 20%에서 나옴. GLD 제거 시 σ_p ≈ 14%로 상승.
3. **꼬리위험 측면** — **정상 시장 안정, 레짐 전환 취약**. 2020-03 COVID급 이벤트에서는 하루 −5% 이상 가능. Long-only 제약으로 본질적 헤지 불가 → 유일한 대응은 **GLD + 현금**.
4. **알파 감쇠 측면** — McLean & Pontiff(2016): "발견된 팩터는 반쯤 죽은 팩터". 1주 지평 모멘텀의 사후적 기댓값은 거의 0. **본 포트폴리오의 가치는 "수익 추구"가 아니라 "리스크 균형"**에 있다.

| 시나리오 | 가정 | 예상 1주 수익 |
|---|---|---|
| Base | 정상 시장 | +0.3% (~$30) |
| Risk-On Rally | SPY +3%, QQQ +4%, GLD −1% | **+2.1%** |

## 5. 자기 평가

### 5.1 예상 수익률 범위

**−2.5% ~ +3.0%** (1주, 중앙값 ≈ +0.3%)

근거: σ_p ≈ 12.5% → 1주 경험적 68% 구간 ≈ ±1.7%, 95% 구간 ≈ ±3.4%. 모멘텀 팩터의 1주 기댓값은 0에 수렴하므로 중앙값은 equity risk premium/52 ≈ +0.1%p 수준. 이는 "정상 시장 지속" 가정에 강하게 의존하며, 실제로는 단일 뉴스 이벤트(FOMC·지표·지정학)가 1주 수익 대부분을 결정할 수 있다.

### 5.2 과제에서 느낀 한계

1. **1주 보유의 근본 한계** — 어떤 모델로도 1주 방향성은 통계적으로 유의하지 않다(6차시 52.2%). 결과의 상당 부분은 **운**이며, **성과로 전략을 평가해서는 안 된다**(7차시 "방법론의 시연이지 성과의 입증이 아니다").
2. **샘플 크기 n=1** — 백테스트·라이브 각 1회. 과정의 일관성이 유일한 평가 기준.
3. **거래비용·슬리피지 미반영** — 실전 10bp+ vs 백테스트 1bp (5차시 p.13). 실전 수익률은 시뮬레이션보다 구조적으로 낮다.
4. **LLM 비결정성·시차** — 같은 입력에 다른 출력 가능, 학습 시점 이후 뉴스 미지. 본 과제는 내가 최신 뉴스를 수동 주입해 대체 → 재현성 불완전.
5. **제약의 비효율** — Long-only·5자산·정수주 제약은 이론 최적 프론티어 대비 수 %p 손실. GLD 20% 목표 = 실제 7주·$1,960로 괴리.
6. **사전 체크리스트 (내가 틀릴 수 있는 지점)**: SPY–GLD 상관 양의 역전, QQQ 고평가, 엔화 강세로 EWJ 훼손, 긴급 FOMC/지정학 이벤트.

### 5.3 배운 것

- **3차시 4-Layer**: L3 제약(Long-only, USO=0)이 L1·L2를 역방향으로 바꾼다 — $10,000 장난감 포트폴리오에서도 동일 원리 작동.
- **5차시 "데이터·피처가 모델보다 중요"**: 5개 피처 + Inverse-Vol만으로도 설득력 있는 비중. 오버엔지니어링 거부.
- **7차시 Bounded Discretion**: LLM을 신탁도 무용지물도 아닌, **역할·제약·JSON 안의 구조화된 보조 분석가**로 운영하는 것이 실용적.

5. Inverse-Vol 초기 비중 → Alpha Blending (α=0.3, β_LLM ∈ [0.8, 1.2])
6. Hard Constraints 적용 (개별 40%, SPY+QQQ 55%, USO 0%)
7. 정수 주 이산화: `shares_i = floor(w_i · 10000 / price_i)`
8. σ_p, β, VaR, 상관행렬 재계산 → 리스크 해석 확정
